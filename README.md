# Siuslaw Tech LLC Website

Static, multi-page website for Siuslaw Tech LLC.

Structure
- index.html
- about/index.html
- business/index.html
- residential/index.html
- assets/css/styles.css
- assets/js/main.js

Git setup
- If this folder already has a git remote, you can commit and push normally.
- If you need to connect a remote repo:
  1. git init
  2. git add .
  3. git commit -m "Initial website"
  4. git remote add origin <your-repo-url>
  5. git push -u origin main

Cloudflare Pages
